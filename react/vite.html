Vite

Vite는 웹 개발을 위한 최신 빌드 도구 중 하나입니다. 웹 애플리케이션의 개발 및 빌드 프로세스를 더욱 빠르고 효율적으로 만들어주는 도구입니다.

Vite는 프로젝트를 위해 미리 번들을 생성하는 대신, 빠른 개발 서버를 통해 모듈을 실시간으로 바로 실행시키는 방식을 채택합니다.
Vite는 Vue.js를 기반으로 만들어졌지만, 다른 프레임워크나 라이브러리와 함께도 사용할 수 있습니다. 최신 JavaScript 모듈 시스템과 브라우저의 네이티브 지원에 의존하여 빌드 프로세스를 간소화하고 개발자들에게 더 나은 개발 경험을 제공합니다.
